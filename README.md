

App theo dõi sức khỏe và dinh dưỡng

## Chức năng

- Đăng ký/đăng nhập
- Ghi chép thức ăn hàng ngày
- Nhận diện món ăn bằng camera (dùng AI)
- Xem thống kê calo và dinh dưỡng
- Lập kế hoạch tập luyện
- Theo dõi lượng nước uống

## Tech stack

**Backend:**
- Node.js, Express
- PostgreSQL + Prisma
- JWT auth

**Frontend:**
- React Native (Expo)
- TypeScript
- React Navigation

**AI:**
- Google Gemini API

## Cài đặt

### Backend
```bash
cd back-end
npm install
npx prisma migrate dev
npm start
```

### Frontend
```bash
cd front-end
npm install
npx expo start
```
